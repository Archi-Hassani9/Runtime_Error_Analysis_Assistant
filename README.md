# Runtime_Error_Analysis_Assistant
This project is a Python-based Runtime Error Analysis Assistant chatbot developed during internship. It help users identify, understand and resolve runtime errors using structured responses.

## Technologies Used
- Python
- Flask
- Groq API
- HTML, CSS, JavaScript

## Features
- Runtime error explanation
- Guided debugging support
- Knowledge base integration
- AI-powered responses
